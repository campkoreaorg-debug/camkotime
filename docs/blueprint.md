# **App Name**: VenueSync

## Core Features:

- User Authentication: Secure login page with separate buttons for admin and viewer roles.
- Admin Dashboard: Dedicated admin dashboard with panels for staff management, schedule management, and map view.
- Viewer Map View: Single-page map view for viewers without password protection.
- Real-time Updates: Utilize websocket so changes made by admin immediately reflected on the viewer's map view in real-time.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a sense of trust and authority, fitting for administrative control and a clean viewer experience.
- Background color: Very light grey (#F0F2F5), a desaturated version of the primary color, creating a clean and unobtrusive backdrop for content.
- Accent color: Warm orange (#FF9800), an analogous hue, for interactive elements and highlighting important updates.
- Font pairing: 'Poppins' (sans-serif) for headlines, offering a geometric and modern feel, paired with 'PT Sans' (sans-serif) for body text, ensuring readability and a slightly warmer touch.
- Responsive design to ensure accessibility across devices.
- Consistent use of recognizable icons across both admin and viewer interfaces for ease of navigation.
- Smooth transitions for a seamless user experience when updating map data and panel transitions.
